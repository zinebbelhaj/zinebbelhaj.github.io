# Zineb Belhaj — Premium Developer Portfolio

A luxurious, modern developer portfolio website that highlights Full Stack capabilities (React, Laravel, NodeJS) through elegant modern layout interfaces blended with cultural Moroccan zellige themes.

## 🎨 Theme Specs
- **Backgrounds**: Rich Black (`#0B0B0B`) & Warm Beige (`#F5E6D3`)
- **Primary Accents**: Gold (`#D4AF37`) & Purple (`#8B5CF6`)
- **Typography**: Outfit (Headings) & Plus Jakarta Sans (Body)
- **Styling**: Vanilla CSS, Glassmorphic panels, and smooth custom canvas particle layouts.

## 🚀 Advanced Interaction Systems
1. **Interactive Skills Orbit**: Core skills (React, Laravel, Node, MySQL, MongoDB, Git) rotate dynamically around a glowing "ZB" badge shield.
2. **Interactive Cursor Glow**: Custom pointer tracking with responsive gold and purple gradients.
3. **Canvas Particles Background**: Performance-friendly floating points.
4. **3D Tilt Project Cards**: Dynamic mouse-tilt calculations and Spotlight gradients mapping cursor location.
5. **Loader Intro Screen**: Rotating Moroccan star logo paths, gold shine, and text reveals.
6. **Animated Statistics**: Dynamic count-ups on view entries.
7. **Accessibility & SEO**: Responsive grid breakpoints, keyboard layouts, semantic outline marks, and valid Open Graph metrics.

## 🛠️ Setup & Local Dev
Open the `index.html` file directly in any modern web browser or serve it locally using a simple web server (e.g. `Live Server` extensions or `http-server` via npm).

To run a quick server using Node's npm:
```bash
npx http-server ./
```
Or with Python:
```bash
python -m http.server 8000
```
Then navigate to `http://localhost:8000`.
